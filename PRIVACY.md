# Privacy Policy / Politique de confidentialité

**Last updated / Dernière mise à jour :** 18 mars 2026

---

## 🇬🇧 English

**Omnivox Dark Theme** is built with privacy and security as top priorities.

1. **No Data Collection:** This extension does not collect, read, transmit, or sell any personal information, browsing history, grades, or any other user data.
2. **Local Storage Only:** We strictly use your browser's local storage (`chrome.storage.local`) to save your theme preference (whether the dark mode switch is turned ON or OFF). This data never leaves your device.
3. **No Third-Party Tracking:** There are no analytics tools, trackers, or remote scripts included in this extension. All code runs locally on your machine.
4. **Permissions:** The host permission for `*://*.omnivox.ca/*` is only used to inject the local CSS stylesheets needed to darken the page interface.

If you have any questions or concerns, please open an issue on the [GitHub repository](https://github.com/majeurbilly/dark_omnivox/issues).

---

## 🇫🇷 Français

**Omnivox Dark Theme** a été conçu en faisant de la vie privée et de la sécurité des priorités absolues.

1. **Aucune collecte de données :** Cette extension ne collecte, ne lit, ne transmet et ne vend aucune information personnelle, historique de navigation, notes scolaires ou toute autre donnée utilisateur.
2. **Stockage local uniquement :** Nous utilisons strictement le stockage local de votre navigateur (`chrome.storage.local`) pour mémoriser votre préférence de thème (si l'interrupteur du mode sombre est activé ou désactivé). Ces données ne quittent jamais votre appareil.
3. **Aucun traçage tiers :** Il n'y a aucun outil d'analyse, traqueur ou script distant inclus dans cette extension. Tout le code s'exécute localement sur votre machine.
4. **Autorisations :** L'autorisation d'accès à l'hôte `*://*.omnivox.ca/*` est utilisée uniquement pour injecter les feuilles de style CSS locales nécessaires pour assombrir l'interface de la page.

Si vous avez des questions ou des inquiétudes, veuillez ouvrir un "issue" sur le [dépôt GitHub](https://github.com/majeurbilly/dark_omnivox/issues).
